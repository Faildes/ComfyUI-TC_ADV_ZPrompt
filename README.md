# TeamC Advanced Z-Image Prompts
Extension that allows to use emphasize and AND on Z-Image

## Usage

1. Put this extension in `ComfyUI/custom_nodes`

2. Under conditioning/advanced, you can find **TeamC Advanced Z-Image Prompts (TC_ADV_ZPrompt)** node

3. Connect *Clip* to the clip, set values as you desired (default recommended, higher the value, stronger the effect gets), put the prompt, and the output to corresponding prompt part
